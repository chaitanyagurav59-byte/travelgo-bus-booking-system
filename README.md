# TravelGo -- Bus Booking System

## 🚌 Overview

TravelGo is a full-stack bus booking web application built using
**Flask**, **PostgreSQL**, **HTML**, **CSS**, **Bootstrap**, and
**JavaScript**. The application enables users to search for buses,
select seats, book tickets, manage bookings, and contact customer
support through a responsive web interface.

------------------------------------------------------------------------

## ✨ Features

-   User Registration & Login
-   Secure Session Management
-   Search Buses by Source & Destination
-   View Bus Details
-   Seat Selection
-   Passenger Details Form
-   Booking Confirmation
-   My Bookings
-   Cancel Booking
-   Contact Us Form
-   PostgreSQL Database Integration
-   Responsive User Interface
